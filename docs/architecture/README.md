# الهندسة المعمارية للنظام (System Architecture)

يحتوي هذا المجلد على الرسومات والمخططات البيانية التوضيحية الخاصة بهندسة النظام، بما في ذلك:
- مخططات تدفق البيانات (Data Flow Diagrams) من ملفات الـ PDF إلى قاعدة البيانات المتجهية.
- رسومات مكونات النظام (Component Diagrams) وعلاقات الحاويات (Docker Containers).
- مخطط العلاقات بين الخدمات: FastAPI Backend ↔ PostgreSQL ↔ Qdrant ↔ Ollama LLM.
- مخططات تسلسل العمليات (Sequence Diagrams) لعمليات البحث السيمانتك والاستشارة القانونية.
