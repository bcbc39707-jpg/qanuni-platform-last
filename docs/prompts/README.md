# قوالب الموجهات (Prompt Templates)

يحتوي هذا المجلد على قوالب الموجهات (System Prompts) المستخدمة في التفاعل مع نماذج الذكاء الاصطناعي (LLM)، بما في ذلك:
- **موجه المساعد القانوني الرئيسي**: التعليمات المرسلة لنموذج Aya/Gemini لتوليد الإجابات القانونية.
- **موجه تحليل العقود**: التعليمات الخاصة بكشف الثغرات والمخاطر في المستندات القانونية.
- **موجه صياغة المستندات**: التعليمات الخاصة بتوليد العقود واللوائح والمذكرات القانونية.
- **موجه التلخيص القانوني**: التعليمات الخاصة بتلخيص النصوص القانونية الطويلة.

## ملاحظة مهمة
- جميع الموجهات يجب أن تتوافق مع قواعد مكافحة الهلوسة المحددة في `.agents/rules/anti_hallucination.md`.
- يجب تضمين تعليمات صريحة بالتوثيق والاستشهاد وفقاً لـ `.agents/rules/citation_rules.md`.
