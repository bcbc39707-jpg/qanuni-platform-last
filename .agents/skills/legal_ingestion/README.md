# مهارة استيعاب وتغذية البيانات القانونية (Legal Ingestion Skill)

تحتوي هذه المهارة على الأكواد والبرامج النصية الخاصة بالوكيل الذكي والمسؤولة عن:
- قراءة وتدقيق ملفات الـ PDF الأصلية.
- استدعاء محرك الـ OCR واستخراج النصوص.
- تحويل النصوص المستخرجة إلى هياكل بيانية (JSON).
- تقسيم وهيكلة المواد وتأكيدها.
- مزامنة ورفع البيانات إلى PostgreSQL وقاعدة بيانات المتجهات Qdrant.
